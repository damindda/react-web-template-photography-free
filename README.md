### This template is focused on Photography Theme

### React + GSAP Free Web Template

- React 19
- Tailwind CSS v4
- GSAP (GreenSock Animation Platform)

## Features

- Fully responsive and mobile-friendly
- GSAP Timelines
- React Motions Effects
- Parallax Effect
- Control ScrollTrigger & ScrollSmoother
- Pinned Style Elements

## How to run this Locally

```bash
# 1. Clone the repo
git clone https://github.com/damindda/react-web-template-photography-free.git

# 2. Install dependencies
npm install

# 3. Run the project
npm run dev
```
